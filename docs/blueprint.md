# **App Name**: ubx

## Core Features:

- Hero: A captivating hero section to immediately grab user attention.
- Features: Detailed features section highlighting the benefits of ubx.

## Style Guidelines:

- Primary color: Rich blue (#4285F4) to convey trust and reliability.
- Background color: Light blue (#E8F0FE) to ensure a calm, trustworthy atmosphere.
- Accent color: A vibrant green (#34A853) to draw attention to conversions and important actions.
- Body and headline font: 'Inter' sans-serif for a clean and professional look.
- Use clear, concise icons to represent metrics and features within the sales page.
- Maintain a clean, structured layout that prioritizes key sales information and CTAs above the fold.
- Subtle animations and transitions to guide the user through the sales funnel.
