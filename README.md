# Unboxx – Lead Handling Tools & AI Automation for Coaching Institutes

Unboxx is a platform designed to help coaching and training institutes improve their lead handling, follow-ups, and admissions using simple tools. Once the process is fixed, Unboxx turns it into a fully automated 24/7 system powered by AI.

## About Unboxx

Most coaching institutes don’t lack leads; they lack a consistent process to handle those leads. Unboxx provides practical tools and templates that help institutes understand where leads are leaking and how to fix their communication flow.

After the process is structured, Unboxx offers an AI automation system that replies instantly to inquiries, sends follow-ups automatically, and helps convert more students into admissions without extra staff.

## What Unboxx Provides

### Free Tools
- Calculators for lead leakage and conversions  
- Follow-up templates and message scripts  
- Checklists and SOPs for counsellors  
- Simple tools to improve communication and structure  

### AI Automation
- Instant responses to new inquiries  
- Automated follow-ups and reminders

- Admission-focused workflows operating 24/7
